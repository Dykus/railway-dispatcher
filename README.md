# ЖД Диспетчерская

Веб-приложение для учёта движения железнодорожных вагонов на промышленной станции.

## Возможности
- Учёт вагонов на путях с визуализацией
- Глобальные и локальные сроки с обратным отсчётом
- История перемещений и архив
- Экспорт данных в Excel
- Ролевая модель доступа (наблюдатель, диспетчер, супервизор, администратор)
- Резервное копирование базы данных
- Журнал действий
- Иконка в системном трее

## Установка и запуск
### Для разработки
1. Клонируйте репозиторий
2. Создайте виртуальное окружение: `python -m venv venv`
3. Активируйте его: `venv\Scripts\activate`
4. Установите зависимости: `pip install -r requirements.txt`
5. Запустите: `python run.py`

### Готовый EXE
Скачайте `RailwayDispatcher.exe` из раздела [Releases](https://github.com/Dykus/railway-dispatcher/releases) и запустите.

## Технологии
- Python 3.8+
- Flask
- SQLite
- Pandas, OpenPyXL
- PyInstaller (для сборки EXE)
